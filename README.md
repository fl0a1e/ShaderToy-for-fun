### 本地开发环境

除了在 ShaderToy 上直接构建，vscode也有一个非常好用的插件

- VS Code 扩展页直接下载 Shader Toy 或去 vscode 官网下载 https://marketplace.visualstudio.com/items?itemName=stevensona.shader-toy

代码页右键 `Shader Toy: Show GLSL Preview`即可实时观察效果
